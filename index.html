trancisiones con css

Permiten cambiar suavemente una propiedad CSS de un estado a otro durante un período de tiempo

Propiedades de las transiciones

  • transition-property: Define qué propiedad cambiar
  • transition-duration: Define la duración del cambio. (s o ms)
  • transition-timing-function: Controla la aceleración o desaceleración
    • linear → Velocidad constante.
    • ease (por defecto) → Comienza lenta, acelera y luego desacelera.
    • ease-in → Comienza lenta y acelera.
    • ease-out → Comienza rápida y desacelera.
    • ease-in-out → Comienza y termina lenta
    • transition-delay: Retrasa el inicio de la transición.

Transicion de color de fondo

a transición de color de fondo en CSS permite cambiar suavemente el color de fondo de un elemento cuando se activa un cambio de estado, como un :hover. Esto crea un efecto visual agradable y fluido que mejora la experiencia del usuario al interactuar con la página. Por ejemplo, al pasar el cursor sobre un botón, su color de fondo puede cambiar de azul a rojo, atrayendo la atención del usuario sin ser abrupto.
